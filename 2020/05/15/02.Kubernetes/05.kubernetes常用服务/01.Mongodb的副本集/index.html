01.Mongodb的副本集

HeiYueQiMa 2020-05-15 07:38:25
Mongodb

一. 基础准备

需要有storgeclass 我这里用的是 cephfs

二. 创建无头service 用于dns的通信

这里将服务和statefulset都放在rocket 命名空间内

kubectl create ns rocket  
apiVersion: v1
kind: Service
metadata:
  name: mongodb-service
  namespace: rocket
  labels:
    service: mongo
spec:
  ports:
  - name: mongo
    port: 27017
    targetPort: 27017
  clusterIP: None
  selector:
    app: mongodb

三. 创建有状态集合StatefulSet

这里申请的pvc绑定的容器路径是 /data/db
其余参数保持默认 后续弄出来升级版本的时候 在添加

apiVersion: apps/v1
kind: StatefulSet
metadata:
  name: mongo-replicate
  namespace: rocket
spec:
  # 这里需要对应service的Metadata里面的name
  serviceName: "mongodb-service"
  replicas: 3
  selector:
    # 这里是选择POD的标签
    matchLabels:
      app: mongodb
    # pod模板
  template:
    metadata:
      labels:
        app: mongodb
    spec:
      terminationGracePeriodSeconds: 10
      containers:
        - name: mongo
          image: mongo:4
          command:
            - mongod
            - "--replSet"
            - rs0
            - "--bind_ip"
            - "0.0.0.0"
            - "--dbpath"
            - /data/db/
          ports:
            - containerPort: 27017
          volumeMounts:
            - name: mongo-persistent-storage
              mountPath: /data/db
  # 申请pvc模板
  volumeClaimTemplates:
  - metadata:
      name: mongo-persistent-storage
    spec:
      accessModes: ["ReadWriteOnce"]
      storageClassName: rook-cephfs
      resources:
        requests:
          storage: 20Gi

暂时没有选择对外开放服务 后续可能会用Tunnel的方式开放服务
