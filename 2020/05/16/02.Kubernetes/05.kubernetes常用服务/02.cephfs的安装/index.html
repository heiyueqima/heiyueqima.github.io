02.cephfs的安装

HeiYueQiMa 2020-05-16 23:26:37
Kubernetes > ceph

ceph是一个很棒的分布式存储 支持3种分布式存储

  • filesystem （文件系统)
  • objectbucket （存储桶）
  • Block Storage (块存储)

我们这里主要用ceph的文件系统 用于容器的数据持久化 这里选择的是 rook-ceph 开箱即用

一. rook-ceph 文档地址

rook-cephfs文档

rook-ceph github

二. rook-ceph 安装

我们安装的也很简单 都是用rook准备的yaml执行一下

git clone https://github.com/rook/rook.git
cd ./rook/cluster/examples/kubernetes/ceph

一般的目录就是我上面命令执行的路径
先安装3个必须安装的yaml

kubectl apply -f common.yaml
kubectl apply -f operator.yaml
kubectl apply -f cluster.yaml

然后安装fs文件系统 还有storageclass

我都是使用的默认参数 只是学习使用 生产环境应该要根据自己的情况进行调整

可以去rook的文档地址 研究一下参数

kubectl apply -f filesystem.yaml
kubectl apply -f csi/cephfs/storageclass.yaml

这样就可以了 其他的就是开放ceph的dashborad了

三. ceph-dashboard 的ingress文件

创建一个ingress用于面板的访问

证书需要提前准备好 我这里就不搞了 

apiVersion: extensions/v1beta1
kind: Ingress
metadata:
  annotations:
    nginx.ingress.kubernetes.io/backend-protocol: HTTPS
  name: ceph.xxx.com
  namespace: rook-ceph
spec:
  rules:
  - host: ceph.xxxx.com
    http:
      paths:
      - backend:
          serviceName: rook-ceph-mgr-dashboard
          servicePort: 8443
  tls:
  - hosts:
    - ceph.xxxx.com
    secretName: cert-ceph

获取后台登录密码

kubectl -n rook-ceph get secret rook-ceph-dashboard-password -o jsonpath="{['data']['password']}" | base64 --decode && echo
